# 🍜 맛집지도 — 사용 설명서

음식 사진을 고르면 사진 속 **촬영 위치(GPS)와 날짜**를 자동으로 읽어서
지도 위에 나만의 맛집 이력을 만들어 주는 앱입니다.

---

## 1. 아이폰에 설치하기 (GitHub Pages 이용, 무료)

아이폰에서 쓰려면 인터넷 주소(https)가 필요합니다. GitHub Pages가 무료이고 가장 쉽습니다.
**코딩 지식 없이 마우스 클릭만으로 가능합니다.**

1. https://github.com 에서 무료 회원가입 (이미 계정이 있으면 로그인)
2. 오른쪽 위 **+** → **New repository** 클릭
   - Repository name: `food-map` 입력
   - **Public** 선택 → **Create repository** 클릭
3. 만들어진 페이지에서 **uploading an existing file** 링크 클릭
4. 이 폴더(`food-map`) 안의 파일 전부를 창에 **드래그해서 업로드**
   - `index.html`, `manifest.json`, `sw.js`, `icons` 폴더 안의 png 3개
   - ⚠️ icons 폴더는 통째로 드래그하면 폴더 구조가 유지됩니다
   - 아래 **Commit changes** 버튼 클릭
5. 저장소 페이지에서 **Settings** → 왼쪽 메뉴 **Pages** 클릭
   - Branch: `main` 선택 → **Save**
6. 1~2분 뒤 `https://내아이디.github.io/food-map/` 주소가 생깁니다
7. **아이폰 Safari**에서 그 주소를 열고
   → 하단 **공유 버튼(⬆️)** → **홈 화면에 추가**
   → 홈 화면에 "맛집지도" 앱 아이콘이 생깁니다! 🎉

> 앱을 수정한 뒤에는 바뀐 파일을 같은 방법으로 다시 업로드하면 됩니다.
> (`sw.js` 안의 `foodmap-v1`을 `v2`, `v3`처럼 올려주면 아이폰에서 바로 갱신돼요)

## 2. PC에서 테스트하기

`실행.bat` 더블클릭 → 브라우저가 열립니다.
(사진 선택 → 위치/날짜 자동 인식까지 PC에서도 동일하게 테스트 가능)

## 3. 사용 방법

| 하고 싶은 것 | 방법 |
|---|---|
| 사진 추가 (과거 사진 소급 포함) | 주황색 **＋** 버튼 → 갤러리에서 음식 사진 여러 장 선택 |
| 별점 매기기 | 지도 핀 또는 목록에서 가게 탭 → 별 탭 |
| 가게 이름/메모 수정 | 가게 화면에서 바로 입력 (자동 저장) |
| 음식명/카테고리 수정 | 가게 화면에서 사진 탭 → 입력 또는 🤖 AI 분류 |
| 검색 | 상단 검색창 (가게명·음식·주소·메모 전부 검색) |
| 카테고리/음식 필터 | 검색창 아래 칩(버튼) 탭 |
| 친구에게 공유 | 가게 화면 → 📤 공유 (카톡, 문자 등으로 전송) |
| 백업 | 설정 → 데이터 내보내기 (JSON 파일 저장) |

### 🤖 AI 자동 분류 (선택 사항)

설정 탭에서 **Anthropic API 키**를 등록하면:
- 사진 추가 시 **음식 사진인지 자동 판별** (음식 아니면 자동 제외)
- **카테고리 + 음식명**을 AI가 자동으로 붙여줌
- 키 발급: https://console.anthropic.com (사용한 만큼 과금, 사진 1장에 몇 원 수준)
- 절약 모드(Haiku)를 선택하면 비용이 약 1/5로 줄어듭니다

키가 없어도 모든 기능을 수동 입력으로 쓸 수 있습니다.

## 4. 알아둘 점 (중요)

- **위치 정보가 있어야 자동 표시됩니다.**
  아이폰: 설정 → 개인정보 보호 → 위치 서비스 → 카메라 → "앱을 사용하는 동안" 확인.
  위치가 없는 사진은 지도에서 직접 탭해서 위치를 지정할 수 있어요.
- **해외 사진도 OK** — GPS만 있으면 전 세계 어디든 지도에 표시됩니다.
- **사진 원본은 저장하지 않습니다.** 작은 썸네일만 앱에 보관하므로 용량 걱정이 없어요.
- **가끔 백업하세요.** 아이폰은 오래 안 쓴 웹앱의 데이터를 지울 수 있습니다.
  (설정 → 데이터 내보내기 → 파일 앱에 저장)
- 처음 사진을 올릴 때 가게 이름은 지도 데이터(OpenStreetMap)에서 자동으로 찾아오는데,
  등록 안 된 가게는 이름이 비어 있을 수 있어요 → 직접 입력하면 됩니다.
